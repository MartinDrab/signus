char* iniparser_getstr

(dictionary* d, char* key)

Get the string associated to a key
This function queries a dictionary for a key. A key as read from an ini file is given as "section:key". If the key cannot be found, a NULL value is returned. The returned char pointer is pointing to a statically allocated string in this module.
Parameters:
d - Dictionary to search
key - Key string to look for
Returns:
pointer to statically allocated character string
