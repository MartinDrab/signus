Installation instructions

Follow these steps to compile the library:

1. Edit the Makefile to indicate the C compiler you want to use, the options to provide to compile ANSI C, and possibly the options to pass to the ar program on your machine to build a library (.a) from a set of object (.o) files.

Defaults are set for the gcc compiler and the standard ar library builder.

2. Type 'make', that should do it.

To use the library in your programs, add the following line on top of your module: 

	#include "iniparser.h"

And link your program with the iniparser library by adding -liniparser.a to the compile line.

See the file test/initest.c for an example.
