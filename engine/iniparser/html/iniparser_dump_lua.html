void iniparser_dump_lua

(dictionary* d, FILE* f)

Dump a dictionary to an opened file pointer.
This function prints out the contents of a dictionary, one element by line, onto the provided file pointer. It is OK to specify stderr or stdout as output files.

The output syntax is fully compatible with lua tables.

Parameters:
d - Dictionary to dump.
f - Opened file pointer to dump to.
Returns:
void
