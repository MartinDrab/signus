int iniparser_setstr

(dictionary* ini, char* entry, char* val)

Set an entry in a dictionary.
If the given entry can be found in the dictionary, it is modified to contain the provided value. If it cannot be found, -1 is returned.
Parameters:
ini - Dictionary to modify.
entry - Entry to modify (entry name)
val - New value to associate to the entry.
Returns:
int 0 if Ok, -1 otherwise.
